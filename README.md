# linux-log-file-analyzer
Linux Log File Analyzer using Grep, Redirection, Aliases and Cron

## Project Overview

This project analyzes Linux system log files using Grep, Aliases, Redirection, Bash scripting, and Cron automation.

---

## Objectives

- Extract error entries from system logs
- Create reusable aliases
- Generate summary reports
- Store reports in a separate directory
- Automate report generation using Cron

---

## Technologies Used

- Ubuntu Linux
- Bash Shell
- Grep
- Cron
- Redirection

---

## Features

- Alias for log filtering
- Error report generation
- Summary report
- Automatic report storage
- Cron scheduling

---
